  • ECPublicKeyComponents

    The components of an EC public key. See RFC-7518, Section 6.2.1

    Note

    To ensure proper JWK JSON encoding, the component data must refer to the unsigned big-endian octet representation of the component’s values, encoded using the minimum amount of octets needed to represent the value.
