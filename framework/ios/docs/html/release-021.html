Release v0.2.1

Release Notes

  • Fixed incompatibility with Swift 5.4 (Xcode 12.5) and above. Our SDK includes a group of binary frameworks, so it should be compatible with following XCode versions going forward.
  • Removed frameworks directory from within Sample App, and linked Sample App to frameworks directory within the release at the top-level (so that we don’t distribute duplicated frameworks files within the Sample App)

Migration Guide

Updating the frameworks

  1. Download frameworks.zip and iosSample.zip and open both.
  2. Open iosSample/iosSample.xcworkspace and Sample App should work linked to the frameworks files from frameworks folder you just opened.
  3. Copy the new frameworks files from the frameworks folder, and overwrite your old frameworks files in your own project.
  4. Clean build folder, delete XCode DerivedData, and re-build.
