Release v0.1.1

Release Notes

  • Feature: added support for bitcode compliance
  • Improvement: removed the need to manually embed the OpenSSL fat framework.
  • Improvement: upgrade of the OpenSSL version to the last stable one, v1.1.180

Migration Guide

CocoaPods updates

Update the OpenSSL dependency in your Podfile as follows:

pod 'OpenSSL-Universal', '= 1.1.180'
