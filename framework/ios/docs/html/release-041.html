Release v0.4.1

Release Notes

This is a micropatch release that applies some minor changes (off of v0.4.0) to allow the SDK to operate independently without including the new Allegion modules (if Allegion is not needed), as well as some build fixes. Please refer to the v0.4.0 release notes for upgrade instructions and a list of major updates.

Updates include:

  • Optional “core” SDK is provided without Allegion modules
  • iOS SDK is compatible with XCode 15
  • Fixes build error related to unneeded Sentry dependency that is not used
