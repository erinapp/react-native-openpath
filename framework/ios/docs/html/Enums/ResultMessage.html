ResultMessage

public enum ResultMessage<T>
extension ResultMessage: Equatable

Represents a message to be sent across the bridge to JavaScript, either a success or failure.

  • message

    This, formatted as an untyped dictionary in the format the SDK uses to report results.

    A success message looks like:

    {
  "data": {
    this.name: this.result
  }
}

    A failure message looks like:

    {
  "err": {
    "message": this.mssg,
    "code": this.code
  }
}

    Declaration

    Swift

    public var message: [String : [String : Any]] { get }
  • successResult

    If this is .success, gets this.result, or if this represents a failure, returns nil.

  • failureResult

    If this represents a failure, returns a typle of (message: this.mssg, code: this.code) or nil if this is a .success
