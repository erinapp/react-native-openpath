iOS SDK Release v0.4.4

Release Notes

  • Added a privacy manifest file in order to comply with Apple’s new requirements documented here: https://developer.apple.com/documentation/bundleresources/privacy_manifest_files/describing_use_of_required_reason_api

  • Upgraded the SDK’s dependencies to versions which either remove usage of required reason APIs or provide privacy manifest files.

  • Fixed some minor bugs in the Allegion functionality.

  • Removed external dependency on XCGLogger

iOS Sample app

  • Update to iOS 16 and new SwiftUI navigation

  • Use Int for requestId instead of Int32

Migration Guide from 0.4.3 to 0.4.4

  • Replace the existing v0.4.3 directory with the contents of this ZIP file.

  • If you had a Cocoapods patch to work around the bug in the AWSIoT library version 2.31.1, that is no longer needed, as that dependency has been updated to a newer version.

  • Refer to other migration guides if you are migrating from an earlier SDK version.
